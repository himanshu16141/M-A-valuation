# Mergers & Acquisitions (M&A) Analysis Project

## Overview
This project presents a detailed analysis of a merger/acquisition, focusing on valuation, strategic rationale, and financial impact.

## Objective
To understand how M&A deals are structured and evaluate their impact on company performance.

## Key Analysis
- Valuation of target company
- Synergy estimation
- Financial impact of the deal
- Strategic rationale behind the acquisition

## Tools Used
- Microsoft Excel
- HTML/CSS
- Financial research

## Conclusion
The analysis highlights how the deal creates value through synergies and strategic alignment.
